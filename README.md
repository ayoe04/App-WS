# App Wrap Station

Aplikasi ini dikembangkan untuk membantu manajemen pesanan dan pembungkus (wrapping) produk.
Aplikasi ini masih awal, masih perlu penambahan dan perbaikan.

## Fitur pada Dashboard
- Input data customer
- Detail inspeksi kendaraan
- Generate laporan customer by pdf

## Cara Menjalankan
1. Clone repository
2. Jalankan `npm install`
3. Jalankan `npm start`

## Aplikasi sudah di deploy : https://app-ws.vercel.app/
